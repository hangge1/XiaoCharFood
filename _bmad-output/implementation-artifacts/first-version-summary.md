# XiaoCharFood 第一个版本实现说明

## 已实现范围

- 微信小程序四个主 tab：`首页 / 做饭 / 餐厅 / 我的`
- 首页：
  - 记录早餐、午餐、晚餐、加餐
  - 添加饮食标签和放纵餐标签
  - 删除饮食记录
  - 最近 7 天低压力周回顾
- 做饭：
  - 需求关键词输入
  - 本地固定菜谱库推荐完整一日菜单
  - 自动生成买菜清单
  - 保存一日菜单计划
  - 将某一餐推荐转为饮食记录
- 餐厅：
  - 新增餐厅记忆
  - 保存菜品、备注、人均、场景标签
  - 搜索餐厅、下拉联想和筛选“还想再去”
  - 搜索无结果时可直接转入记录新餐厅
- 我的：
  - 设置做饭偏好
  - 查看本地数据统计
  - 清空本地数据
- 数据层：
  - `utils/store.js` 统一封装本地存储
  - `utils/recommender.js` 统一封装本地规则推荐

## 关键实现决策

- 第一个版本本地优先，不依赖后端和云开发。
- 做饭建议使用确定性本地规则，不接 AI 或第三方菜谱源。
- 当前菜谱先使用 40 道常见中餐和轻食的固定菜谱库，避免未接 AI 时推荐结果过空。
- 餐厅记忆只做私人记录，不做公开点评和地图探索。
- 所有健康相关文案保持低压力表达，不做诊断或惩罚式反馈。

## 已知限制

- 当前项目没有 `package.json` 或 `miniprogram-ci`，无法在命令行完成微信小程序编译。
- 需要用微信开发者工具打开项目做最终编译和真机/模拟器验证。
- `project.config.json` 当前 `libVersion` 是 `trial`，正式上传前建议在微信开发者工具中固定基础库版本。
- 第一个版本没有云同步、家庭共享、图片上传存储和 AI 菜谱生成。

## 2026-07-10 体验优化

- TabBar 文案从功能名调整为更有家庭生活感的 `饭桌 / 今晚 / 回味 / 小家`。
- 饮食记录新增 `time` 字段，记录列表展示日期 + 具体时间。
- 新增首次称呼设置，首页问候会带昵称；`我的` 页面也可修改称呼。
- 做饭推荐增加多关键词输入：哪一餐、口味、已有食材、场景约束综合排序。
- 餐厅页重构为明确的两个模式：`找去过的店` 和 `记录新餐厅`，降低搜索框歧义。
- 清理默认模板的 `logs` 入口和启动日志逻辑。

## 2026-07-10 视觉氛围优化

- 新增本地 TabBar PNG 图标资产：饭碗、锅、回味标记、小家。
- 全局视觉从普通表单改为温暖饭桌感：浅暖背景、软卡片、圆角按钮、冰箱贴式 sticker。
- 首页改为“饭桌营业中”氛围，突出私人饭桌记录和今日小贴纸。
- 做饭页改为“今晚灵感小厨房”，推荐卡片更像候选菜单。
- 餐厅页改为“回味雷达”，搜索和新增分别有明确视觉语义。
- 我的页改为“小家冰箱贴”，弱化设置页感，强调饭搭子偏好和本地私密记录。

## 2026-07-10 一日菜单推荐优化

- 做饭页从“单餐推荐”调整为“一日菜单推荐”，一次输出早餐、午餐、晚餐、加餐。
- 取消手动选择哪一餐，降低做饭前规划成本。
- 新增需求关键词输入，支持混合输入口味、已有食材和目标倾向。
- 快捷关键词包括：清淡、省事、家常、川菜、减肥、增肌、高蛋白、少油、微辣、牛肉、鸡蛋、豆腐。
- 推荐结果会汇总买菜清单，方便提前准备食材。
- 当前使用 40 道常见中餐和轻食组成固定菜谱库，覆盖家常、川菜倾向、清淡、减肥、增肌、高蛋白、快手等关键词。
- 减肥、增肌仅作为生活化饮食偏好关键词处理，不输出专业健康或医疗建议。

## 2026-07-10 回味地图能力优化

- 餐厅新增时支持通过微信地图选择餐厅地址，并保存经纬度。
- `app.json` 增加 `scope.userLocation` 权限说明，用于展示附近 5km 去过的餐厅。
- 回味页新增 `附近地图` 模式，获取当前位置后显示 5km 内保存过地址的餐厅。
- 地图使用小程序 `map` 组件和 markers 标注餐厅位置。
- 附近列表按距离排序，并可调用 `wx.openLocation` 打开餐厅位置。
- 未保存经纬度的历史餐厅仍保留在普通搜索列表中，但不会出现在附近地图。

## 2026-07-10 回味搜索与菜谱库优化

- 回味页的“找回餐厅”增加显式搜索按钮，降低只有输入框带来的操作歧义。
- 输入店名、菜品或备注时展示下拉联想结果，点结果即可快速定位。
- 搜索无结果时展示明确空状态，并可直接把当前关键词带入“记录新餐厅”。
- 搜索结果摘要统一由 JS 生成，避免没有最近到店记录时展示异常。
- 做饭页快捷关键词调整为更贴近真实需求的 `清淡 / 省事 / 家常 / 川菜 / 减肥 / 增肌 / 高蛋白 / 少油 / 微辣 / 牛肉 / 鸡蛋 / 豆腐`。
- 菜谱推荐器增加自然关键词映射，例如川菜、粤菜、家常菜、清爽、备餐等。
